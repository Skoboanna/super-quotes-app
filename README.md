# super-quotes-app
Reading the quotes from json file, adding new quotes to json file

## Requirements

* chalk
`npm install chalk`
* yargs
`npm install yargs`


## Start
* From src:
`node app.js`

## Available commands

* app.js __quotes__
  * Prints all quotes from the list
* app.js __author__ *name* *surname*
  * Prints all quotes by provided author
